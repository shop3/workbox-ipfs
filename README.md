# Workbox IPFS

IPFS router for a Workbox service worker.

## Installation

```bash
npm install --save workbox-ipfs
```

## Usage

```js
import { initialize } from 'workbox-ipfs';

initialize({
  /* IPFS Options */
})
```

## Install Dev Environment

```bash
npm install

npm run husky:install
```

## Run Dev Application

```bash
npm run develop
```
